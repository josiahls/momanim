# momanim
> This project is in early / playground development. Everything is subject to breaking.
A Mojo implementation of the 3Blue1Brown tool.

## Installation

[Install pixi](https://pixi.prefix.dev/latest/installation/)

```bash
# Install openh264 locally and run tests. 
pixi run -e use-openh264 test_all
```

## Developer Notes
If working in cursor, you can change the editor to use cursor instead.
```bash
git config core.editor "cursor --wait"
```
